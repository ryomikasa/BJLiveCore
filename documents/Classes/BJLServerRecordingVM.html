– requestServerRecording:

开始/停止云端录课

- (nullable BJLError *)requestServerRecording:(BOOL)on

Parameters

on

YES：打开录制，NO：关闭录制

Return Value

BJLError: BJLErrorCode_invalidCalling 错误调用，如在非上课状态下调用此方法； BJLErrorCode_invalidUserRole 错误权限，要求老师或助教权限。

Discussion

老师才能开启录课，参考 BJLErrorCode_invalidUserRole

上课状态才能开启录课，参考 roomVM.liveStarted

此方法需要发起网络请求、检查云端录课是否可用

  • 如果可以录课则开始、并设置 serverRecording
  • 否则发送失败通知 requestServerRecordingDidFailed:
